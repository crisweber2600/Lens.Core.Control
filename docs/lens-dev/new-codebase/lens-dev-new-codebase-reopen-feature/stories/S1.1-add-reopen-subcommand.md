# S1.1 â€” Add `reopen` Subcommand to `feature-yaml-ops.py`

## Context

This story implements the core command contract for governed feature reopen. It adds a `reopen` subcommand to `feature-yaml-ops.py` that transitions a terminal feature (`complete`/`archived`) back to an active express phase, updates `feature.yaml`, and syncs `feature-index.yaml`.

## Target File

```
TargetProjects/lens-dev/new-codebase/lens.core.src/_bmad/lens-work/skills/lens-feature-yaml/scripts/feature-yaml-ops.py
```

## Command Contract

```
feature-yaml-ops.py reopen
  --feature-id <id>           # feature identifier
  --governance-repo <path>    # path to governance repo root
  --to-phase <phase>          # default: expressplan
  [--actor <label>]           # audit label, default: "system"
```

## Acceptance Criteria

1. **Terminal guard:** Only features with `phase` in `{complete, archived}` OR `status` in `{complete, archived}` are allowed to reopen. All others return error JSON `{"status": "error", "reason": "reopen_not_allowed"}` with exit code 1.
2. **Phase target validation:** `--to-phase` must not itself be a terminal phase. If track is null or missing in `feature.yaml`, emit a warning but continue (bypass-allowed with warning).
3. **`feature.yaml` mutations (applied atomically):**
   - `phase = <to-phase>`
   - `status = active`
   - Remove `completed_at` key (unconditionally; safe since terminal features always have it)
   - Append entry to `phase_transitions`: `{from: <old_phase>, to: <to-phase>, actor: <actor>, timestamp: <ISO-8601>}`
4. **Index sync:** Call existing `sync_feature_index` helper; output JSON includes `index_synced: true`.
5. **Output:** On success, return JSON `{"status": "ok", "feature_id": ..., "new_phase": ..., "index_synced": true}`.

## Implementation Notes

- Reuse existing path-resolution helpers (`resolve_feature_path`, `resolve_governance_repo`) already in the file.
- `phase_transitions` is a YAML list in `feature.yaml`; if key is absent, initialize as empty list before appending.
- Use `datetime.utcnow().isoformat() + "Z"` for timestamp.
- Return non-zero exit code on all error paths.
