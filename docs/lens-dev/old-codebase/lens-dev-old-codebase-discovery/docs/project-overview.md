## Executive Summary

`lens.core.src` is the published module core for the LENS Workbench platform. It is a meta-framework that delivers lifecycle orchestration assets, AI agent skills, prompts, operational scripts, and embedded documentation that enable structured software development workflows inside AI coding assistants.

This is not a web service or UI application. It publishes lifecycle assets and operational tooling through a controlled release promotion pipeline.

The project has a single authoritative source layer:

- `_bmad/lens-work/` is the canonical module source — skills, prompts, scripts, tests, templates, contracts, and embedded docs.

## Key Capabilities

- Publishes the Lens command surface through module prompts and skills.
- Defines the `lens-work` lifecycle contract, registries, and module metadata.
- Ships installation, bootstrap, validation, PR, and preflight tooling.
- Carries a deep built-in documentation set under `_bmad/lens-work/docs/`.
- Promotes source changes through a controlled release workflow.
